安装和更新snippet

您只需要几分钟的时间就可以将博客主题切换到snippet，感受snippet的美轮美奂。

安装前准备

如果你在此之前使用的是 Hexo 2.x 版本，为了避免未知的错误，请备份好数据，或者建立新的博客目录

环境搭建

Node.js 环境、Git 环境以及 Hexo ,如果你尚未安装或者不了解 Hexo，请参考官方教程进行了解以及安装。如果需要构建工具请自行安装，或使用本主题的Gulp方式。

安装依赖

snippet主题依赖有ejs 模版引擎 、 Less CSS预编译语言、本地搜索和官方插件

$ npm install hexo-renderer-ejs --save
$ npm install hexo-renderer-less --save
$ npm install hexo-deployer-git --save
$ npm install hexo-generator-search --save

下载主题

有两种方式下载主题--下载 *.zip 文件和通过 git ，无论哪种方式请确认下载的文件目录和博客站点的theme子目录中的landscape在同一目录中

从Github上获取代码：

$   git clone git@github.com:wiliam2015/hexo-theme-snippet.git themes/snippet

更新主题

更新主题代码

$ cd themes/snippet
$ git pull
